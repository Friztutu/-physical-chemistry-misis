О проекте

Актуальность

  • Задачи решаемые в курсе безопасности жизнидеятельности применяются на реальном производстве.
  • Каждая задача состоит из большого количества трудоемких расчетов, в связи с чем намного проще алгоритмизировать процесс ее решения
  • Реализация проекта в виде сайта имеет ряд значительных плюсов, таких как:
    • Отсутствие необходимости устанавливать сторонее ПО на устройство
    • Доступ из любой точки мира и на любом девайсе
    • Мы не собираем никаких личных данных своих пользователей

Технологии

  • Python - Объектно-ориентированный язык программирования с множеством библиотек
  • Html и CSS - Оформление и внешний вид страниц нашего веб-ресурса.
  • Bootstrap - это набор инструментов для создания сайтов и веб-приложений. Именно он позволил в короткие сроки написать полностью адаптивный сайт.
  • Postgresql - Для хранения результатов расчетов
  • Django - Фреймворк для веб-приложений на языке Python, использующий шаблон проектирования MVC(Model-View-Controller)
  • Django ORM- Позволяет нам взаимодействовать с базой данных, используя код Python, а не SQL. Да и это, наверное, главная и мощная особенность фреймворка Django
